Just shipped ByteBrief – my latest side project that I've been working on in evenings and weekends.

It's a tech news aggregator that pulls the latest articles from TechCrunch, The Verge, and Wired all in one place. Simple idea, but I wanted something that actually worked the way I needed it to.

What I built:
• Google sign-in (Firebase Auth)
• Save articles for later with one click
• Filter by topics I'm actually interested in
• Dark mode because... why not

The whole thing is a static site deployed on GitHub Pages. No complex backend – just Firebase doing what it does best.

Been using it myself for the past couple weeks and it's been nice having everything in one feed instead of jumping between three different sites. The bookmark feature alone has been a game changer for articles I want to read but don't have time for right now.

Took longer than I expected (isn't that always the case?), but pretty happy with how it turned out. Next up: adding more sources and maybe some customization options.

Check it out if you're interested – would love to hear what you think!
